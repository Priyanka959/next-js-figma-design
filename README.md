# design-figma-2

A Next.js (App Router) UI kit / design implementation based on Figma — a simple marketing/site frontend prototype.

## Features
- Next.js 13+ with App Router
- TypeScript-ready layout
- Tailwind CSS integration
- Components for home, contact, and shared UI
- Photo gallery and lightbox dependencies included

## Prerequisites
- Node.js 18+ and npm 

## Install
```bash
npm install
```

## Development
Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```



## Project structure (selected)
- `src/app/` — Next.js app routes and pages
- `src/components/` — Reusable UI components (home, contact, shared)
- `public/assests/` — Static assets
- `package.json` — Scripts and dependencies
